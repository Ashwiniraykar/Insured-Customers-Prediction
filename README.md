# Insured-Customers-Prediction
Predicting the Amount of Money Spent on Insured Customers
To predict the dollar amount of money spent by the insurance company on insured customers, based on a sample data that consists of the profile of insured customers. 

##Insured Customers' Data
The insured customers' data is in a csv file. It has information sconsisting of:
age
sex (female, male)
BMI
Children
Smoker (yes, no)
Region (northeast, northwest, southeast, southwest])
expenses
The value we want to predict is expenses
